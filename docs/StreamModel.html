Class: StreamModel

StreamModel()

Streaming model abstract class.

Constructor

new StreamModel()

Interface (abstract class) for stream models. This class provides a template that the stream models must follow in QStream (based on scikit-multiflow). The class should not be instantiated as it is an abstract class.

Parameters:
Name Type Default Description
class_label int null If given it means the node is a leaf and the class label associated with it is class_label (default: null).
split_att_index int null If given it means the node is an inner node and the split attribute is split_att_index (default: null).
split_att_value int null If given it means the node is an inner node and the split value is split_att_value.

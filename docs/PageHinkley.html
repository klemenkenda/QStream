Class: PageHinkley

PageHinkley(min_num_instances, delta, lambda, alpha)

Page-Hinkley drift detector.

Constructor

new PageHinkley(min_num_instances, delta, lambda, alpha)

Page Hinkley change detector (based on scikit-multiflow implementation, which is further based on the book - Gamma, Knowledge Discovery from Data Streams, 2013, pp. 76) This change detection method works by computing the observed values and their mean up to the current moment. Page Hinkley won't output warning zone warnings, only change detections. The method works by means of the Page Hinkley test. In general lines it will detect a concept drift if the observed mean at some instant is greater then a threshold value lambda.
Parameters:
Name Type Default Description
min_num_instances int 30 The minimum number of instances before detecting change.
delta float 0.005 The delta factor for the Page Hinkley test.
lambda int 50 The change detection threshold.
alpha float The forgetting factor, used to weight the observed value and the mean.
